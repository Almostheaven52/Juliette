This is an improved version of the timer. It is animated like the original but this one changes color from green to orange to red the closer you get to zero. Instead of a scroll wheel to enter the values you have input boxes. These boxes allow you to type precise inputs. The ability to scroll through the numbers is still there but it has been mapped to buttons along the right edge of the textbox to make it more precise.
